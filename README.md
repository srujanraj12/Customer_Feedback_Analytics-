# Customer_Feedback_Analytics- This interactive Power BI Dashboard transforms customer feedback into clear and colorful insights. Dive into the data, track customer sentiments, and spot areas for improvement.

Here are some of the key tasks performed :

Visualized: Crafted an interactive dashboard using Power BI and Excel, transforming data into meaningful insights.

Monitored: Tracked Key Performance Indicators (KPIs) for evaluating and enhancing customer satisfaction.

Empowered: Enabled data-driven decision-making, optimizing strategies based on feedback analysis.
